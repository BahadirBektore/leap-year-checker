# Leap Year Checker

Leap Year Checker is a simple and interactive web application that checks whether a given year is a leap year or not.  
Built using HTML, CSS, and JavaScript.

---

## 🌐 Live Demo

Try it here: [https://bahadirbektore.github.io/leap-year-checker/](https://bahadirbektore.github.io/leap-year-checker/)

---

## 🚀 How to Use

1. Enter a year.  
2. Click the "Check" button.  
3. The result will be displayed on the screen:  
   - “Leap Year” → The year is a leap year  
   - “Not Leap Year” → The year is not a leap year

---

## 🛠 Technologies

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
